# Proximity Alarm

A proximity alarm inspired by the Proximity Alarm from the game "Raimbow Six Siege"

## Components
- Microcontroller
- HC-SR04 Ultrasonic Proximity Sensor
- NPN Transistor
- Active Buzzer
